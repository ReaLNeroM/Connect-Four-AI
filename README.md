# Connect-Four-AI
The popular board game Connect Four implemented in C++ using SFML graphics. By default, the program has an AI hint that's implemented using Monte Carlo Tree search that utilizes multi-core processing.

# Requirements:
- G++ with C++14 support
- SFML libraries
- pthread

# How to run?
make all
run ConnectFour
